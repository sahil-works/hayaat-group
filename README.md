# Hayaat Group — Next.js Website

A full multi-page Next.js implementation built from Figma designs.

## Tech Stack
- **Next.js 14** (App Router), **TypeScript**, **Tailwind CSS**
- **Fonts**: Antic Didone (headings) + Inter Light (body) via Google Fonts

## Pages
| Route | Page |
|---|---|
| `/` | Home |
| `/about` | Who We Are |
| `/what-we-do` | What We Do |
| `/investments` | Investments / Real Estate |
| `/news` | News |

## Getting Started
```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # Production build
```

## Design Tokens
- `#6b6355` warm taupe background
- `#3d3830` dark sections
- `#4a4438` footer
- `#bfaf9d` gold accent/text
- `#f5f2ee` cream/light sections

## ⚠️ Image URLs
Figma asset URLs expire in 7 days. Replace them with your own images in `/public/images/` and update the `src` values in each component.
